同类基金竞品分析器

一条命令出竞品分析报告

TL;DR

输入一只基金代码，工具自动选 8 只同类竞品、抓取业绩与风险指标、横向比较，再调用大模型生成一份带定位分析的 Markdown 报告与 Excel 对比表。原本半天的活，现在三五分钟。

痛点 · PROBLEM

做产品对接渠道、写一页通、判断新基金定位时，第一步永远是同类竞品分析——要查 8 到 10 只同类型基金的费率、规模、近一/三年业绩、最大回撤、夏普比率等指标。

在传统流程里，这意味着：打开 Wind 或同花顺，一只一只搜，复制粘贴到 Excel，手动算对比项，再写一段定位分析。一只基金做下来半天，做完才发现要再加几只竞品，又得重来。

更要命的是，下次再分析另一只基金，整个流程几乎要重复一遍——价值含量低，但占用大量时间。

怎么做的 · HOW IT WORKS

工具用 Python 写成，输入只要一个基金代码。流程分四步：① 用 akshare 接口拉取基金基础信息和同类清单，自动按类型+规模匹配 8 只竞品；② 抓取所有候选基金的业绩、风险、规模、费率数据；③ 生成 Excel 对比表，每列最优值高亮成绿色；④ 把对比表喂给大模型（默认 Kimi，可切 DeepSeek），让它写一段定位分析。

几个工程上比较得意的细节：本地缓存——同一只基金当天再跑只要几秒；自动选竞品——指数型按跟踪标的、主动型按类型+规模匹配；「复核提醒」——大模型分析里凡是出现表格外的数字，会自动打上 ⚠️ 标记，提醒人工核对，防止 AI 编数据。

$ python3 main.py --code 007994
→ output/007994_20260612/report.md + comparison.xlsx（约 3 分钟）
真实输出 · OUTPUT
report.md · 竞品对比表 + 大模型定位分析
comparison.xlsx · 横向对比，绿色 = 该列最优
成果 · OUTCOME

完成竞品分析从「半天手工」到「三分钟自动」的跨越，输出物从一张表升级为对比表 + 定位分析报告双件套。

已成为日常做新基金研究、写产品材料前的第一道工序。
